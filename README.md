# shemoo-store

## التشغيل

1. ثبّت الحزم:

```bash
npm install
```

2. عدّل ملف `.env.local` وضع بيانات Supabase الصحيحة.

3. شغّل المشروع:

```bash
npm run dev
```

## ملاحظات

- المشروع يستخدم Next.js App Router.
- تم إعداد alias `@/` داخل `tsconfig.json`.
- تم إضافة الإعدادات الأساسية اللازمة لتشغيل Tailwind و TypeScript.
